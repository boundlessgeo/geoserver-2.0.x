CSS Styling Extension
---------------------

This extension allows the user to style map layers using a CSS-like styling
language.

Installation
------------
1. Stop GeoServer
2. Copy the jar files contained in this zip into geoserver/WEB-INF/lib
3. Restart GeoServer

Usage
-----

Visit the Web UI and log in.  The extension adds a link to the sidebar entitled
"Map CSS Demo" which allows editing CSS styles with a map preview updated live.
More information on the CSS syntax and features is available in the user
manual: http://docs.geoserver.org/stable/en/user/community/css/index.html
